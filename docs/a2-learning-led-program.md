# A2 learning-led operating program

Status: active from 2026-07-14. This is the repeatable operating checklist; the roadmap owns product
direction and the backlog owns implementation status.

## The five windows

1. `reisen-verkehr` + `einkaufen-reklamation`
2. `gesundheit-arzttermin` + `arbeit-beruf`
3. `nebensaetze-plaene` + `biografie-erfahrungen`
4. `freunde-feste` + `lernen-verstehen`
5. `aemter-dienstleistungen` + A2 checkpoint

Before opening either topic, audit its mission coherence, A2 ceiling, natural German, accepted
renderings, distractors, audio purpose, explanatory feedback, and model → scaffold → fade → transfer
sequence. Preserve topic, outcome, set, item and card identities; increment an item revision when its
task or grading contract changes.

After each pair, run `bun run progress:audit --profile vitaly`. Record response-mode distribution,
delayed and overdue probes, productive (`x-de`) lapses, persistent focus errors, linguistic-review
candidates, structured writing/speaking use, session workload, the pair table and pilot feedback.

The 2026-07-13 queue of 29 rejected production renderings was reviewed before this program opened.
No rendering was added merely because it was grammatical in isolation: the apparent candidates
changed definiteness (`den` → `einen`), communicative meaning (`wo ist` → `wie komme ich`) or
asserted a narrower physical state (`ist` → `steht`); the remaining rows were lexical, case,
word-order or spelling errors already handled by the authored contract. Revisit only when a new,
revision-matched attempt supplies evidence.

## Decision rules

- Repeated plausible rejected German triggers answer-key review, not automatic acceptance.
- A drill requires errors across multiple items plus recent or probe evidence.
- Two failures on one competence/confusion trigger lesson or drill review; one triggers inspection.
- Two productive-direction lapses on one lexeme trigger review of its forms, contrast and context.
- Optional content, document viewing, reference lookup and receptive Wortfeld entries never become
  mastery evidence or cards.

## Calendar and B1 gates

On or after 2026-08-02, review the A1 21-day cohort. If ordinary probe load still fits the
three-per-session cap, pilot listening probe families only for `termine-vereinbaren` and
`reisen-verkehr`, observe a complete 2/7/21-day cycle, and only then consider expansion.

A2 closes only after the checkpoint and its 2/7/21-day evidence are reviewed. Freeze B1 identities
after that review, not before. Browser TTS remains the fallback unless evidence isolates audio
quality—not language difficulty—as the comprehension limit.
