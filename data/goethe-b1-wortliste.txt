# Goethe-Zertifikat B1 — Wortliste coverage manifest.
# Headwords only (the official glosses and examples are copyrighted; all
# translations and examples in content/vocab/ are original). One headword per
# line, bare form matching the vocab `de` field. The first sections mirror the
# PDF's own Wortgruppen (pp. 8-15); the rest keeps the alphabetical list
# (pp. 16-102) in per-letter sections, so completeness stays checkable against
# the PDF page by page. A word listed in several sections counts once, for its
# first.
#
# The only ~ marks here are INHERITED: a B1 headword carries a ~ iff the same
# word is ~-marked in the A1 or A2 manifest — the course already teaches it as
# grammar and the no-flashcard decision is frozen there (a plain line would
# undercount the head start and invite a future B1 deck to card grammar the
# course teaches without cards). Every inherited ~ still pays the taught-surface
# check (`bun run validate`). B1-only words get NO ~ yet: no B1 content exists
# to earn one, and their card-vs-grammar classification happens when B1 units
# are planned. Two near-matches deliberately stay plain: B1 `lieb` (A2's
# ~lieb- is the letter salutation stem; B1 lists the standalone adjective — a
# different word, card-worthy) and B1 `geehrt` (A2's ~geehrt- earns via the
# inflected `geehrte`; the bare form is not in the taught surface, so the ~
# would be unearned).
#
# Extracted from the official PDF (Goethe-Institut / Universität Freiburg/
# Schweiz / ÖSD, 2016, "circa 2400 lexikalische Einheiten"); every headword
# below occurs verbatim in it, except two PDF typos corrected on the evidence
# of their own example sentences ("der Hinweise, -e" -> Hinweis;
# "die Kursleiter, -nen" -> Kursleiterin).
# Notes on normalization (same rules as the A2 manifest, plus the trinational
# variants B1 is full of):
#  - paired entries are split ("der Absender, -/die Absenderin, -nen" ->
#    Absender + Absenderin);
#  - collapsed entries are expanded ("(ab)fahren" -> fahren + abfahren,
#    "Kilo(gramm)" -> Kilo + Kilogramm, "Mathe(matik)" -> Mathe + Mathematik);
#  - D/A/CH cross-references each contribute their word ("die Treppe (D, CH)
#    -> A: Stiege" -> Treppe + Stiege), listed once at first occurrence;
#  - spelling variants of one word keep the Duden-recommended form (Erdgeschoss
#    not Erdgeschoß, Fantasie not Phantasie, schick not chic, Müsli not Müesli,
#    Ketchup not Ketschup, Ski not Schi, Soße not Sauce, gern not gerne);
#  - determiner/adjective stems keep the PDF's form (all-, ander-, irgendein-);
#  - obligatory reflexives keep sich (sich beeilen); parenthesized "(sich)" is
#    stripped (duschen, rasieren);
#  - genuine multi-word units stay verbatim (zu Fuß, ein bisschen,
#    entweder … oder);
#  - the PDF's date/clock format demos and digit-only material carry no
#    headword and are not listed.
